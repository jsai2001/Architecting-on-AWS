# Architecting on AWS - Course Overview

## Course Description

Architecting on AWS is designed for solutions architects, solution-design engineers, and developers seeking comprehensive understanding of AWS architecting principles. The course focuses on building resilient, secure, and highly available IT solutions on the AWS Cloud using the AWS Well-Architected Framework.

### Course Format
- Recorded instruction modules
- Multimedia and interactive content
- Pre and post assessments
- Real-life scenario-based learning
- Hands-on solution building practice

### Key Focus Areas
- Account security
- Networking
- Compute resources
- Storage solutions
- Database services
- Monitoring and automation
- Containers and serverless architecture
- Edge services
- Backup and recovery

## Learning Objectives

Upon completion, you will be able to:

- **Security & Best Practices**
  - Identify AWS architecting best practices
  - Summarize account security fundamentals
  - Build secure virtual networks with private/public subnets

- **Architecture & Compute**
  - Build multi-tier architectures in AWS
  - Select appropriate compute resources for business use cases

- **Storage & Databases**
  - Compare AWS storage products and services
  - Contrast AWS database services based on business needs

- **Operations & Monitoring**
  - Implement monitoring, load balancing, and auto scaling
  - Utilize AWS automation tools for infrastructure management

- **Networking & Connectivity**
  - Design hybrid networking solutions
  - Implement network peering and gateway solutions

- **Modern Architecture Patterns**
  - Deploy AWS container services
  - Leverage serverless services for business benefits
  - Implement edge services for latency and security

- **Resilience & Recovery**
  - Design backup and recovery solutions
  - Ensure business continuity and resiliency

## Target Audience

- Solutions architects
- Solution-design engineers
- Developers learning AWS architecting
- AWS Solutions Architect-Associate certification candidates

## Prerequisites

### Required Experience
- Minimum one-year experience with visual analytics
- Completed AWS Technical Essentials course

### Technical Knowledge
- Distributed systems concepts
- General networking principles
- IP addressing fundamentals
- Multi-tier architecture patterns
- Cloud computing basics

## Important Notes

> **Service Availability**: AWS Cloud9, AWS CodeCommit, and AWS Snowball Edge are no longer available to new customers. Existing customers can continue using these services normally.
